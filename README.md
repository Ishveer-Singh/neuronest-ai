# Neuronest AI 🚀

Neuronest AI is a modern AI-powered study helper landing page built using HTML and CSS.  
The project is designed to help students learn smarter with AI-based tools like doubt solving, note generation, summaries, and study planning.

## ✨ Features

- AI Study Helper UI
- Responsive Design
- Modern Dark Theme
- Smooth Navigation
- Feature Cards Section
- Contact/Profile Section
- Clean Footer & CTA Design

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript *(future integration)*
